\## 📌 Commit Convention (Emoji Guide)



| 타입 | 이모지 | 코드 | 설명 | 예시 |

|------|--------|--------|--------|--------|

| feat | ✨ | `:sparkles:` | 새로운 기능을 추가할 때 사용합니다. | ✨ feat: 로그인 폼 유효성 검사 추가 |

| fix | 🐛 | `:bug:` | 버그를 수정할 때 사용합니다. | 🐛 fix: 로그인 버그 수정 |

| style | 💄 | `:lipstick:` | UI/스타일 관련 변경 (기능 변경 없음) | 💄 style: 네비게이션 바 디자인 수정 |

| refactor | 🎨 | `:art:` | 기능 변화 없이 코드 구조 개선 | 🎨 refactor: 컴포넌트 상태 관리 로직 단순화 |

| perf | ⚡ | `:zap:` | 성능 개선 | ⚡ perf: 이미지 로딩 시간 최적화 |

| test | ✅ | `:white\_check\_mark:` | 테스트 코드 추가/수정 | ✅ test: 버튼 컴포넌트 단위 테스트 추가 |

| docs | 📝 | `:memo:` | 문서만 변경될 때 | 📝 docs: README 설치 단계 업데이트 |

| chore | 🔧 | `:wrench:` | 패키지 업데이트 등 일반 작업 (코드 변경 없음) | 🔧 chore: 종속성 패키지 업데이트 |

| revert | ⏪ | `:rewind:` | 이전 커밋 되돌릴 때 | ⏪ revert: "로그인 폼 유효성 검사 추가" 되돌림 |

| init | 🎉 | `:tada:` | 프로젝트 초기 설정 | 🎉 init: 프로젝트 초기 세팅 |

| delete | 🔥 | `:fire:` | 파일/코드 삭제 | 🔥 delete: 사용하지 않는 컴포넌트 삭제 |

| wip | 🚧 | `:construction:` | 작업중(WIP) / 실험적 변경 | 🚧 wip: 새로운 인증 방식 실험 중 |
